What's new in 1.3.3

New
• Usage redesigned: Today now breaks down hour by hour, and the 30-day view is a calendar heatmap
• Type @ in the message box to quickly insert a file from your project
• Sub-agent runs appear as cards with live status and an expandable report
• Preview and export session files — diffs and documents — straight from the Changes view

Fixes & improvements
• AskUserQuestion: free-text answers are now typable
• "Sending…" no longer gets stuck — a delivery times out cleanly, shows an undelivered state, and can be resent
• Tap a file path in a transcript to open it
• Codex session titles no longer show setup blocks or raw IDs

Note: some fixes need the computer-side cc-pocket-daemon on 1.3.3 — run "cc-pocket-daemon update" (or brew upgrade / scoop update).
