What's new in 1.3.6

New
• Scheduled prompts: long-press send to queue a prompt for later (one-off or daily) — great for bedtime tasks. Manage them under Settings.
• Auto-continue after a usage limit: when a turn hits your limit, one tap schedules "continue" for the moment the window resets
• Gateway model presets: one-tap DeepSeek / GLM / Kimi / Qwen / MiniMax when you run Claude Code through a third-party gateway
• Skills & plugins browser on desktop: see what's installed on the computer, with each one's description and details
• View more files: open any file in the working directory from your phone, and larger Office files (xlsx / docx / pptx)

Fixes & improvements
• Big reconnect-stability overhaul for weak and cross-border links: far fewer drops, and reconnects are near-instant instead of replaying the whole history
• Get notified when a turn is waiting for your approval, hits an error, or runs out of usage — not just when it finishes
• Frozen streaming after a mid-turn reconnect now recovers on its own
• iOS "add video" now reads your photo library instead of Files
• Desktop question card no longer shows up empty
• "Load earlier messages" no longer gives up on a slow connection
• /clear now resets the context % badge

Note: some features need the computer-side cc-pocket-daemon on 1.3.6 — run "cc-pocket-daemon update" (or brew upgrade / scoop update).
