本次更新（1.3.6）

新功能
• 定时任务：长按发送把提示词排到稍后执行（一次性或每日），睡前排活很好用，设置里可管理
• 撞限额自动续跑：回合撞到用量上限时，一键把「继续」排到限额重置的那一刻自动执行
• 网关模型预设：走第三方网关跑 Claude Code 时，DeepSeek / GLM / Kimi / Qwen / MiniMax 一键选用
• 桌面端 skills 与插件浏览页：查看电脑上已安装的 skill 和插件，含各自的介绍与详情
• 查看更多文件：手机上打开工作目录里的任意文件，以及更大的 Office 文件（xlsx / docx / pptx）

修复与优化
• 弱网与跨境链路的重连稳定性大幅优化：掉线明显减少，重连接近秒开（不再全量重放历史）
• 回合等你审批、报错、或撞限额时也会推送，不再只在完成时通知
• 回合中途重连后卡住的流式输出现在会自动恢复
• iOS「添加视频」改为读取相册，不再走「文件」
• 桌面端提问卡片不再显示为空
• 「加载更早消息」在慢连接下不再放弃
• /clear 后上下文百分比角标会归零

提示：部分功能需要电脑端 cc-pocket-daemon 同步更新到 1.3.6——运行 cc-pocket-daemon update（或 brew upgrade / scoop update）。
