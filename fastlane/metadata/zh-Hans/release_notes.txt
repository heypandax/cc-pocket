本次更新（1.2.0）

新增
• Token 用量看板：今日消耗、请求数、花费与缓存命中一目了然，附近 7 / 30 天趋势和按模型分布（设置 → Usage），Claude 与 Codex 的用量都会统计
• 会话内切换模型：Opus / Sonnet / Haiku 一键切换并保留完整对话历史；Codex 会话支持 gpt-5.1-codex 系列
• 只看一个助手：同时用 Claude 和 Codex？设置里可以只显示其中一个的会话
• 安装引导：配对页新增电脑端安装引导，macOS / Windows / Linux 各一条命令装好
• 从列表顶部输入路径，直接在任意文件夹开启会话

修复与优化
• 接管电脑端会话不再产生重复会话；接管后电脑端也能看到手机上聊的内容
• 上下文百分比更准确：支持 1M 上下文窗口；接续旧会话立即显示所用模型
• 重连更顺滑：切后台再回来不再闪「重连中」；换网络后自动恢复，无需重启
• 草稿按会话独立保存，重启 App 也不丢
• Windows 电脑：目录树可正常进入子目录，会话接续与 Codex 会话列表恢复正常
• Codex 会话改为显示真实 token 占用，不再套用错误的百分比
• 修复局域网模式下偶发的「API Error 400」与易断线问题

提示：部分功能需要电脑端 cc-pocket-daemon 同步更新到 1.2.0（brew upgrade / scoop update / 重跑安装命令均可）。
