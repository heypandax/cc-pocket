如何在手机上使用 Claude Code

目前有两条实用途径：Anthropic 官方 Remote Control，以及同时支持 Claude Code 和 OpenAI Codex 的开源 CC Pocket。

2026 年 7 月 13 日核验引用官方来源
直接结论

只使用 Claude Code、偏好第一方体验，先选官方 Remote Control；需要同时控制 Claude Code 与 Codex、检查源代码或自托管中继，选 CC Pocket。

方案一：Claude 官方 Remote Control

Anthropic 官方指南给出两种入口：在已有 Claude Code 会话执行 /remote-control，或用 claude remote-control 启动。该指南注明需要 Claude Code 2.1.51 或更高版本，并面向 Pro、Max、Team 与 Enterprise 方案。

方案二：CC Pocket

CC Pocket 在你的电脑上运行 daemon，由它启动或恢复本机 Claude Code CLI；手机或桌面客户端可查看流式输出、发送消息、检查改动，并允许或拒绝工具调用。

  1. 安装 CC Pocket 手机或桌面客户端。
  2. 在运行 Claude Code 的电脑上安装 daemon。
  3. 配对设备，选择项目，再启动或恢复 Claude 会话。

最新安装命令见 项目 README。

安全与限制

CC Pocket 用 P-256 ECDH、HKDF 和 AES-256-GCM 保护客户端到 daemon 的内容，中继只转发密文。加密不会把 daemon 变成沙箱：它仍以本机操作系统用户权限运行；全自动模式可能执行无需逐步确认的命令。

CC Pocket 是独立项目，不隶属于 Anthropic，也未获得其背书。
